Vega Community, Welcome

About Vega

Vega Protocol is a decentralized network, which has been optimized for derivatives trading. Trading on Vega takes place by pseudonymous parties, and there is no requirement to pass KYC or prove your identity to access markets.

Vega is composable with all layer one blockchains. It currently implements a bridge to Ethereum, allowing ERC20 tokens and Ether to be used as collateral for trading. In future, Vega will also implement bridges to Terra, Bitcoin, Polkadot, and much more.

The Vega blockchain uses Tendermint for the consensus layer, and currently processes a new block every ~800ms. The network is capable of processing 1000s of transactions per second.

Blockchain transactions are free on Vega, and users only pay fees when their transaction is matched with another trader. This is consistent with how fees work on centralized exchanges.
